# 1. PROMPT GUIDELINES

## 1.1 MidJourney v7 Best Practices

### Standard Parameters for All Prompts

```
--v 7 --ar 2:3 --q 2 --s 250
```

| Parameter | Value | Purpose |
|-----------|-------|---------|
| `--v 7` | Version 7 | Latest MidJourney model |
| `--ar 2:3` | Portrait ratio | Character reference standard |
| `--ar 16:9` | Landscape | Environmental portraits |
| `--ar 1:1` | Square | Social media / headshots |
| `--q 2` | Quality 2 | Maximum detail |
| `--s 250` | Stylize 250 | Balanced style/accuracy |

### Style Keywords (Include in Every Prompt)

```
cinematic portrait, film still, natural lighting, 
Emmanuel Lubezki cinematography, Bradford Young skin tones,
authentic Dominican, warm color palette, 4K detail
```

### Lighting Keywords by Scene Type

| Scene Type | Lighting Keywords |
|------------|-------------------|
| **Dawn/Morning** | golden hour, warm backlight, soft amber, magic hour |
| **Midday/Game** | harsh natural sun, high contrast, honest light |
| **Interior/Home** | practical warm light, window light, amber tungsten |
| **Evening/Sunset** | orange golden light, rim light, silhouette potential |
| **Stadium/Night** | cool stadium lights, warm skin contrast, dramatic |

### Skin Tone Guidance

For authentic Dominican representation:
```
warm brown skin, healthy glow, natural texture,
properly exposed dark skin, Bradford Young lighting style
```
